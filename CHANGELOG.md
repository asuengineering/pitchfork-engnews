# Change Log

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/)
and this project adheres to [Semantic Versioning](http://semver.org/).

## Version 1.1

- ADD: Create redirection mu-plugin to point posts from Full Circle to their new URL. All post URLs have changed based on new permalink structure. See doc comments in `/inc/redirect-from-fullcircle.php` for details.
- FIX: Heading for "related people by school" in in topic archive page should be sentance case.
- FIX: Include new card formatting for "In the media" section of `asu_person` taxonomy term pages.
- FIX: Links to external media stories now open in the current browser window.
- FIX: External media cards now have separate labels for each topic.
- ADD: Additional logic for retrieving valid alt text from the social media images associated with `/external` news stories.

## Version 1.0

- NEW: Official launched of product to support new ASU Engineering news site. See [ASU Engineering News](https://news.engineering.asu.edu) for working examples of the included features.
